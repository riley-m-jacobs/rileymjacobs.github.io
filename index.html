Outlines:

At this point you are probably somewhat confused how I will combine all these things. Hold tight. We are only on the first step. To start we just need to get outlines of the image of Peqa. I'm new to this stuff so I am going to try and manually do it in Inkscape and GIMP. Let's see how this turns out.

First I put the image into GIMP and outlined it.

Then I outlined it again in a thicker paint tool and filled in some of the features.

After this, I removed the background and did some smoothing.

Next, I put the image into Inkscape so I could get a preliminary SVG file (This is the file type I need for the CAD software). To do this I traced the bitmap of the PNG file from GIMP, simplified the image manually by deleting some of the vector anchor points in the SVG output, then exported the simplified outline.

CAD

TBH the process of getting the outline took a long long time... much longer than I expected. Mad respect to artsy people who do this sort of graphic design stuff daily. At the start of this project I thought I was going to make multiple designs, but alas time is limited so I settled with just just doing this one of Peqa. Now that my aside is out of the way, back to the main topic of this section: CAD.

First I imported the SVG into a sketch in Fusion 360.

Beautiful! Then I did some editing to fix some things and to decide which features to actually use, extruded the outline, and extruded some of the details to a lesser extent.

Nice! At this point you can probably guess what it is. Cookie cutters! I figured with Christmas coming soon, you could make the most unique Christmas cookies ever, depicting the one and only Peqa.

3D Printing

Now for the fun part! For 3D printing I had access to a Sindoh 3DWOX which makes it pretty easy to go from an STL file (exported from Fusion 360) to finished project with only a couple steps.

The first step is to load the STL file into a slicing software in order to slice the model into gcode. Then from there you can save the gcode file to a usb and plug it right into the printer.

4 hours later, we are done!

Reflections

After looking at the finished project, I realized that cookie cutters usually don't have a solid back to them to allow for you to press the cookie out if it get stuck. Unfortunately I could not figure out how to nicely do that given the input of the files I used so instead I printed again with a small hole in the back panel so you can fit your finger through to press something out, however, its probably in your best interest to use non-stick spray or something to help make sure the cookie comes off cleanly.

I also designed the cookie cutter to have two layers, one which would cut out the outline of the design and one which would merely indent a design into the surface of the cookie. I was not sure how deep I should make the indentation design relative to the outline cutter so I varied the relative depths across the two cutters I made (if it doesn't work with either let me know and I can print another one with more empirically driven measurements).

Lastly, it has been a while since I have made cookies, but if I remember correctly lots of cookies start in a small sphere and spread out into circles.
